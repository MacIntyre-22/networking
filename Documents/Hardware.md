## About
So far my home network consist of just a few parts, excluding my personal devices (which you can see in the topology). I plan to add more devices like mini computers for example, and will update this repo as I add to it.

## Server
My main server is an old buisness desktop computer that I had collected at a previous job. 

## Router/Managed Switch
**TP-Link ER707-M2**

My router is connected to my modem and allows me to control every device I want to hook up to the internet without touching the rest of the house. It also hides my devices from the main network and acts as a separate LAN so every device in my setup is ran through this. 

## Unmanaged Switch
**TP-Link TL-SG108**

Connected to my managed switch, all personal devices run through here.
